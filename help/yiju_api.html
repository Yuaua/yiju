一句·接口
很多朋友希望有类似一言(Hitokoto)的古诗词的句子API，于是就写了这篇一句API调用说明。 一句API返回的内容为随机一句古诗词，暂时没有分类等更多功能，后续开发……

另外一句API目前不保证完全稳定，请勿用于严肃用途！

2017.7.2 全站启用HTTPS，但并不强制HTTPS，将以下接口链接中http修改为https即可通HTTPS请求接口！
希望在您的APP、网站等项目内使用时注明API来自一句。


直接返回纯本文

http://yiju.ml/api/word.php
JSON调用(2018.08.23新增)
http://yiju.ml/api/word.php?m=json
返回JSON数据示例：
{"content":"有谁知我此时情，枕前泪共阶前雨，隔个窗儿滴到明。","source":"聂胜琼《鹧鸪天·别情》"}
content：诗句内容 ｜ source：诗句作者
html+js方式调用
在您想显示的地方加入以下代码：

显示作者：
<script type="text/javascript" src="http://yiju.ml/api/word.php?m=js"></script><script>document.write("<div>");yiju(true);document.write("</div>");</script>
